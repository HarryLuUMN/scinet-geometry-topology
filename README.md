# Science Network Geometry and Topology

Working paper draft and reproducible analysis for:

**The Economic Value of Knowledge Geometry: Topological Opportunities and Breakthrough Innovation in Scientific Networks**

The repository expects the prepared OGBN-MAG/MAG dataset at `MAG_Dataset/`. The dataset is intentionally ignored by Git.

## Reproduce

```bash
make analysis
make paper
```

or:

```bash
make all
```

Main outputs:

- `paper/main.pdf`
- `paper/figures/`
- `paper/tables/`
- `data/derived/paper_features.csv`
